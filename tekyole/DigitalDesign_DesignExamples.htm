- Implement an AND gate with a multiplexer?
Connect one of the inputs to the select line of the multiplexer and connect the first input of the multiplexer to 0 and the second one to 1.
-----

- How to make an XOR gate with a multiplexer?
Connect one of the inputs to the select line of the multiplexer. Connect the second input to the first input of the multiplexer and the complement of the second input to the second input of the multiplexer.
-----

- Design a full adder using multiplexer
In a full adder we have 3 inputs (2 one bit inputs and a carry in) and we have two outputs (a one bit output and a carry out). The output (Sum) is one when only one of the inputs is one or when they are all one. It is 0 when all of the inputs are one or when two of the inputs are one. For carry out, it is one when more than one of the inputs is one, and it is 0 when all inputs are zero, or only one of the inputs is 1.
