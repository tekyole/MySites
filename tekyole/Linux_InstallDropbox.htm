Dropbox is a popular cloud storage and online backup system. If you are not familiar with dropbox, there is alot you are missing! check this link out to learn more about dropbox.
Installing dropbox on linux like most of other softwares is not straight forward. In this post I will show you, how you can install dropbox on a linux centOS machine.

Installing dropbox on Linux CentOS
Open Terminal and Switch User to root:
su
When prompted enter root password
Since dropbox uses a package called libgnome, we should install libgnome first:
sudo yum install libgnome
Now you should download dropbox rpm package file from www.dropbox.com. If you are logged in to dropbox in your browser, you should logoff first to find these files.
Download the Fedora (.rpm) file. Depending on your system, select the 64-bit or 32-bit one.
Now in your Terminal, you need to go to the folder that dropbox rpm file is saved. It is usuall downloaded to /home/usr where usr is your username. You can go there using:
cd /home/usr
Install the downloaded rpm file using rpm command.
Keep in mind that dropbox may change the name of the file, as it releases new updates for its software. I downloaded the 64-bit version from dropbox and the filename was: nautilus-dropbox-2015.02.12-1.fedora.x86_64.rpm. Note 2015 in the file name, it is a clear indication that the file name will for sure change when they release new version.
In my case I used this command, you should replace the rpm file name with whatever the name of the rpm file you have downloaded is:
sudo rpm -ivh nautilus-dropbox-2015.02.12-1.fedora.x86_64.rpm
Once you are done with the previous step, click on the "Applications" usually located on top right of your desktop, and look for dropbox. In my case, I found it under internet tab.
dropbox should start to download...
Once the download is completed, you should log in, assuming that you already have an account with dropbox. You should receive an email from dropbox letting you know that your user has install dropbox on a new computer (a security notice)
Click on "open my dropbox folder"

dropbox should automatically start to download your files.
When I was done with installing dropbox, I noticed it has added a (lamp like) icon down the desktop.
