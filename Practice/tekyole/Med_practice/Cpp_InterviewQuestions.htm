What is object oriented programming?
In object oriented programming the data, variables, and functions that perform a certain task are combined into single unit. This unit is called an object. Each object can have properties and member functions. You can call member function to access data of an object.
-----

What is data encapsulation? What are public, private, and protected keywords in C++ used for?
Data encapsulation (sometimes called data hiding) is the mechanism that enables the programmer to control and restrict the access to some of the object's members or components.
In C++, the concept of data encapsulation is supported through the use of the 3 keywords used in declaration of the class:
- public:
Anything in the class that is placed after the public keyword will be accessible to all of the users of the class.
- protected:
Anything in the class that is placed after the protected keyword will only be accessible to the member functions and friends of that class, and to the member functions and friends of derived classes.
- private:
Anything in the class that is placed after the private keyword will only be accessible to the member functions and friends of that class.
-----

Explain the concept of inheritance in object oriented programming.
It allows the code to be reused as many times as needed. The base class once defined and once it is compiled, it need not be reworked. Saves time and effort as the main code need not be written again. Inheritance allows one class to reuse the state and behavior of another class. The derived class inherits the properties and method implementations of the base class and extends it by overriding methods and adding additional properties and methods.
