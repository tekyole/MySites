- Write an algorithm that will get a string and reverses the characters in it
We can use a temporary char variable to store the characters whose their location will be changed:
  void reverse_a_string(char* s) {
    int i;
    char temp;
    size_t l = strlen(s);

    for (i = 0; i < (l/2); ++i) {
      temp = s[i];
      s[i] = s[l - i];
      s[l - i] = temp;
    }
  }
